# leaflet-polyline-mixin

> SCSS mixins for [FCOO/leaflet-polyline](https://github.com/FCOO/leaflet-polyline)

## Description

SCSS functions to create classes for different colors for polyline created using [FCOO/leaflet-polyline](https://github.com/FCOO/leaflet-polyline)


## Installation
### bower
`bower install https://github.com/FCOO/leaflet-polyline-mixin.git --save-dev`

## Usage

In any scss-file in the `\src` directory include

	@import "../bower_components/leaflet-polyline-mixin/dist/leaflet-polyline-mixin";


### mixin

    //lpl- classes
    @include polyline-classes( $name, $color: blue, $border-color: white )
